# gulp

```shell
npm install --save-dev gulp-sass gulp-clean-css gulp-concat gulp-uglify-es gulp-imagemin gulp-rename gulp-plumber
```

- gulp-sass: sass 转为 css
- gulp-clean-css: 压缩 css
- gulp-concat: 合并文件
- gulp-uglify-es: 压缩 js
- gulp-imagemin: 压缩图片
- gulp-rename: 重命名
- gulp-plumber: 错误处理
<!-- - gulp-watch: 监听文件变化 -->
